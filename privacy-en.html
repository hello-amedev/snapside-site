SnapSide Privacy Policy

Last updated: May 18, 2026

SnapSide ("this extension") is a Chrome browser extension developed by ame_dev ("the developer"). This policy explains how user data is handled when you use this extension.

1. Collection of Personal Information

This extension does not collect any personal information. Your browsing history, page content, and input data are never sent to the developer's server.

2. Information Stored Locally

This extension stores the following information only inside your browser (via Chrome's chrome.storage API):

  • Side window settings such as width ratio and position
  • URLs registered as favorites
  • Initial install timestamp (used for managing the free trial period)
  • Session information (main / side window IDs, tab history)

This data persists across browser and PC restarts but is never transmitted externally.

3. Third-Party Services

3.1 ExtensionPay (License management / Payment processing)

License purchases (after the free trial) are processed by ExtensionPay (operated by Glench LLC) using Stripe.

  • When making a purchase, you are redirected to the ExtensionPay payment page (extensionpay.com).
  • Your email address and payment information (credit card details, etc.) are received and processed directly by Stripe. The developer does not have access to this information.
  • For more details, please refer to:
    • ExtensionPay Privacy Policy
    • Stripe Privacy Policy

3.2 Google Favicons API (Favicon retrieval for favorites)

To display the favicon of URLs you register as favorites, this extension sends only the domain name of the URL to Google's public favicon service (https://www.google.com/s2/favicons). This is Google's standard favicon retrieval feature and does not include any personal information.

4. Purpose of Permissions

PermissionPurpose
tabsCreating, moving, and managing tabs (split, merge, swap)
windowsCreating and positioning side windows
storageSaving settings and favorites
contextMenusAdding items to the right-click menu
scriptingRe-injecting content scripts into existing tabs after extension updates
system.displayRetrieving display information for sub-monitor support
host_permissions: <all_urls>Displaying the bottom-left toolbar on all web pages inside the side window

These permissions are used only for providing SnapSide's features and are not used to record or transmit your browsing data.

5. Analytics / Tracking

This extension does not use any analytics tools such as Google Analytics. We do not collect any operation logs or usage statistics.

6. Children's Privacy

This extension is not intended for use by children under the age of 13.

7. Changes to This Policy

This policy may be changed without prior notice. Significant changes will be announced through extension updates.
